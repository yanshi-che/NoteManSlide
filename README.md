### 概要
このプログラムは譜面制作ツールを内包した、自作のリズムゲームです。多少制限がありますが、自分の好きな曲で譜面を作って遊ぶことができます。ノーツは普通のノーツ、ロングノーツ、スライドノーツの三種類があります。

### ポイント
ポイントは何といっても譜面制作ツールがついている点です。個人製作ですので一人で譜面を供給するのは時間的に厳しいですが、ツールがあることで遊ぶ人自身が譜面を作ることができます。本ツールを使用して作った譜面に関しては、好きなようにしていただいて構いません。

### 配布用のURL
https://drive.google.com/drive/folders/1-mc6oSaPd1Zm6_EJpxCKpwbrBwJFBstd?usp=sharing

### 対応している音楽ファイル
- mp3(非推奨、曲によっては途中で途切れてしまいます。)
- wav
- ogg
  
### 出力する譜面データ
- noteman(編集用ファイル)
- json(ゲーム用ファイル)

### 遊ぶ前の下準備
- 配布されているファイルには二次配布等の規約の関係上、BGMのファイルがありません。まずは、「bgm」フォルダ内に好きな音楽のファイルを入れてください。私は魔王魂様のBGMサイバー42のoggファイルを使用しました。
- 同じくサンプル曲のファイルがありません。「musicData」フォルダ内の各フォルダには「music」があります。「Devil's spirit fight battle song07」フォルダの「music」フォルダには「魔王魂 旧ゲーム音楽 戦闘曲07.ogg」(https://maou.audio/category/game/game-battle/page/5/) を、それ以外の「sample」フォルダには「魔王魂 旧ゲーム音楽 戦闘曲メドレー01.mp3」(https://maou.audio/category/game/game-battle/) をそれぞれ魔王魂様のサイトよりダウンロードしてください。

### 曲の追加の仕方
- 作った譜面は曲とともに「musicData」フォルダに追加していきます。ディレクトリ構成は以下のようになっています。  
musicData ---- someThing ---- music  
　　　　　　　　　　　　|  
　　　　　　　　　　　　| ----- saveData  
　　　　　　　　　　　　|  
　　　　　　　　　　　　-- score ---- easy  
　　　　　　　　　　　　　　　　　|  
　　　　　　　　　　　　　　　　　|- normal 
　　　　　　　　　　　　　　　　　|  
　　　　　　　　　　　　　　　　　|  
　　　　　　　　　　　　　　　　　-- hard  
                 
1. 「musicData」フォルダ直下に好きな名前のフォルダを作ります。  
2. そのフォルダの直下に「music」「score」「saveData」フォルダを、さらに「score」フォルダの直下に「easy」「normal」「hard」フォルダをつくります。
3. 「music」フォルダには音楽ファイルを、「score」フォルダの直下のファイルには各難易度に対応した譜面データを入れます。それぞれ名前は何でもいいです。また、「saveData」フォルダは空で構いません。
                                      
### 操作方法
- レーンの左から「D」「F」「G」「H」「J」「K」となっております。スライドノーツは青色が「R」「T」、赤色が「U」「I」がそれぞれ左向き、右向きに対応しています。

### クリア方法
- ゲージが70%以上でクリアとなります。また、ミスなしでFullChain、全てパーフェクトでPerfectFullChainとなります。
                                     
### 譜面の作り方
1. ホーム画面から「NoteEdit」を選びます。
2. メニューバーの「ファイル」から新規作成を選びます。
- 使う音楽ファイルのパス、タイトル、アーティスト、譜面のレベル、BPM、曲の時間、曲が始まるまでの誤差を入力していきます。それぞれ例にあるようなフォーマットに従ってください。例えばBPM200の場合は200.0と入力します。
3. クオンタイズの変更(注意：設置されているノーツは撤去されます)
- 全体を変更したい場合は「クオンタイズ（全体）」から、一小節変更したい場合は左側に小節番号がありますので、それをクリックしてから「クオンタイズ（小節）」で変更してください。
4. ノーツを設置
- 初期設定は普通のノーツです。設置したい場所をクリックしてください。
- ロングノーツは始点だけ拍線をクリックして、伸ばしたいところまでマウスを持って行った後に離してください。
- スライドノーツは横方向に伸ばしてください。
5. テストプレイ
- メニューバーの「テスト」からテストプレイができます。
6. 編集用ファイルの保存
- 「ファイル」の「保存」から編集用のファイルが保存できます。以降は「開く」からそのファイルと音楽ファイルを選ぶだけで作業に復帰できます。
7. ゲーム用ファイルの保存
- 「ファイル」の「エクスポート」 から実際にゲームで遊ぶようのjsonファイルが保存できます。
